## 视图状态说明

### 状态 A: 阵营聚合视图（默认）

```
        ┌──────┐
    ┌───│ 曹魏 │───┐
    │   │ 119人│   │
    │   └──────┘   │
┌───┴──┐       ┌───┴──┐
│ 东汉 │       │ 蜀汉 │
│  62人│       │ 121人│
└───┬──┘       └───┬──┘
    │   ┌──────┐   │
    └───│ 东吴 │───┘
        │ 119人│
        └──────┘

- 每个势力是一个大节点
- 节点大小按人数比例
- 连线粗细按跨阵营关系数量
- 点击某阵营 → 进入展开视图
```


## 交互说明

| 操作 | 效果 |
|------|------|
| 鼠标悬停节点 | 高亮该节点 + 直接关联节点和连线，其他变淡 |
| 点击节点 | 打开右侧详情面板 |
| 双击阵营节点（聚合视图） | 展开该阵营内部人物 |
| 拖拽空白区域 | 平移画布 |
| 滚轮 | 缩放画布 |
| 拖拽节点 | 移动该节点位置 |
| 点击详情面板中的人物名 | 跳转到该人物（图谱居中 + 打开其详情） |
| 点击详情面板中的事件名 | 面板切换为事件详情 |
| 时间轴拖动 | 筛选对应时间段的人物和事件 |
| 搜索框输入 | 实时模糊匹配，选中后定位到该人物 |
| 过滤面板勾选 | 实时过滤图谱中显示的节点 |
| [← 返回总览] | 从展开视图回到阵营聚合视图 |
| [清除过滤] | 重置所有过滤条件 |
