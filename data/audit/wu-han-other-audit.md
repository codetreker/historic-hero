## 六、其他问题

### 6.1 ID 命名不一致

- `zhu-huan-232` 的 name 是 "朱宣"，ID 中含 "huan" 与名字不匹配

- `wu-` 前缀的 ID（如 `wu-tao-huang`、`wu-quan-shang`）与无前缀的同名人物重复

### 6.2 汉献帝重复条目

`han-xiandi`（汉献帝）和 `liu-xie`（劉協）疑为同一人的两个条目，虽然 `han-xiandi` 的 name 是"汉灵帝"对应的是灵帝，而 `liu-xie` 是献帝。需确认：
- `han-lingdi` = 汉灵帝(156-189) ✓
- `han-xiandi` = 汉献帝(181-234) ✓
- `liu-xie` = 劉協(181-234) — 这就是汉献帝本人，与 `han-xiandi` **重复**

### 6.3 简繁体混用

数据中部分 name 使用简体（如曹魏阵营人物），部分使用繁体（如大量 han/wu/jin/other 的后批次人物），建议统一。
