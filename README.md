# Historic Hero · 历史英雄谱

中国历史人物关系可视化 — 三国篇

## 简介

交互式可视化中国历史人物之间的关系、重大事件。MVP 聚焦三国时期（东汉末年 ~280 年），覆盖 500-600 位历史人物。

## 特性

- 🗺 **交互式关系图谱**：基于 AntV G6 的力导向图，按阵营分色分组
- 🔍 **搜索与过滤**：按阵营、身份、关系类型筛选
- 📖 **知识百科**：人物详情（生平、关系、参与事件）+ 事件详情
- ⏳ **时间轴导航**：按时间维度浏览历史演进
- 📱 **响应式设计**：PC / 平板 / 手机适配

## 技术栈

- React + Vite + TypeScript
- AntV G6 v5（图谱渲染，WebGL）
- Ant Design（UI 组件）
- Cloudflare Pages（部署）

## 数据来源

所有人物、关系、事件数据从互联网公开历史资料采集整理，确保历史准确性。

## 开发

```bash
npm install
npm run dev
```
